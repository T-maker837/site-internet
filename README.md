# 📡 PhoneTracker

Suivi GPS en temps réel via navigateur.

## 🚀 Accès rapide
- **Dashboard** : https://VOTRE_USERNAME.github.io/phonetracker/
- **Tracker**   : https://VOTRE_USERNAME.github.io/phonetracker/tracker.html

## ⚙️ Setup
1. Remplir `config.js` avec votre config Firebase
2. Ouvrir `tracker.html` sur les téléphones à suivre
3. Suivre en temps réel sur le dashboard

## 🛠️ Technologies
- Firebase Realtime Database
- Leaflet.js + OpenStreetMap
- Geolocation API (GPS natif)
- Nominatim (geocoding)
